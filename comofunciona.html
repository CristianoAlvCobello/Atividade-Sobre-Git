Basico para usar Git

Configure seu Git.

Agora que você instalou o Git em seu computador, chegou a hora de configurar seu nome de usuário e email do GitHub. Isso é necessário para que cada commit que você fizer seja associado a essas informações. Um "commit" no Git é uma ação que grava as alterações feitas no código em um repositório. Dito isto, vamos lá.

Abra o terminal e digite os seguintes comandos:

  • git config --global user.name "Seu Nome"
  • git config --global user.email "Seu Email"

Principais comandos do Git.

  • "git init": Inicializa um novo repositório Git.
  • "git status": Verifica o status das mudanças no repositório.
  • "git add [arquivo]": Adiciona mudanças no arquivo ao índice.
  • "git commit -m [este é um exemplo de mensagem]": Comita as mudanças adicionadas com uma mensagem descritiva
  • "git push": Envia commits locais para um repositório remoto.
  • "git pull": Atualiza o repositório local com as mudanças do repositório remoto
  • "git commit --amend": Corrige uma mensagem errada em um commit no Git
